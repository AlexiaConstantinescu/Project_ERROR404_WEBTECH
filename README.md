# ASE Notes Manager

Obiectiv
Aplicație web care permite studenților să gestioneze notițele de la cursuri și seminare.

Tehnologii
- Frontend: React.js
- Backend: Node.js + Express.js (REST API)
- Bază de date: PostgreSQL (Sequelize ORM)
- Autentificare: Google OAuth2 (@stud.ase.ro)
- Deployment: Render / Vercel / Railway
